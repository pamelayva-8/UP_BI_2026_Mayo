# Descripción de la Actividad

El objetivo de esta actividad es aprender el uso básico de herramientas fundamentales para proyectos de análisis de datos y desarrollo de aplicaciones interactivas, específicamente:

- GitHub como plataforma de control de versiones y colaboración.
- Streamlit para la creación de aplicaciones web interactivas en Python.

Durante esta práctica se realizó la configuración de un repositorio en GitHub y la ejecución de una aplicación sencilla utilizando Streamlit.

---

# Herramientas Utilizadas

## GitHub
GitHub es una plataforma basada en Git que permite almacenar, administrar y compartir proyectos de software. Además, facilita el trabajo colaborativo y el control de versiones.

### Funciones utilizadas:
- Creación de repositorio
- Subida de archivos
- Actualización de versiones
- Gestión básica del proyecto

---

## Streamlit
Streamlit es una librería de Python que permite crear aplicaciones web interactivas de manera rápida y sencilla, especialmente enfocadas en ciencia de datos e inteligencia de negocios.

### Características principales:
- Fácil integración con Python
- Desarrollo rápido de dashboards
- Visualización de datos interactiva
- Ejecución local mediante navegador
